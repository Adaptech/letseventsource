Recipe Reviews - No tech jargon

Requires an envelope with events in random order - some are business, some are technical. One per table group. Use the EMD in recipe review example. Print it with https://gogs.adaptechsolutions.ca/robert.reppel/emd-pdf.

Howto:

“Take two minutes: Read the requirements. Then take a look at the events in the envelope. Try to arrange them in the right order.”

Participants will run into difficulties putting the technical events in sequence and will start discussing implementation details.

“That was painful. Try something else: Remove the technical events and try now.”

Participants will notice that there are a lot less technical conversations and more focus on business objectives if the technical events are not present.
